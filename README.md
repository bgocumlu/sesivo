# sesivo pages scaffold

Minimal GitHub Pages scaffold for `bgocumlu/sesivo`.

Includes:

- `index.html`
- `docs.html`
- `404.html`
- `robots.txt`
- `sitemap.xml`
- `sitemap.txt`
- `llms.txt`
- `site.webmanifest`

```text
https://sesivo.app/
```
